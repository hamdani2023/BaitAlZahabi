# 🏛️ Bait Al Zahabi — دليل الإصدار v2

## ✅ التعديلات المُنفَّذة في هذا الإصدار

### 1. الترجمة تعمل على كل الصفحات
- تم إضافة `LangWrapper` في `layout.jsx`
- عند تغيير اللغة من أي صفحة، يتغير اتجاه الموقع (RTL/LTR) وكامل النصوص فوراً
- أيقونة الترجمة موجودة في: الرئيسية، القائمة، النادل، الحساب، التقييم، لوحة المدير، لوحة رئيس النُدُل، واجهة النادل

### 2. اسم المطعم ثابت بالإنجليزية
- `RESTAURANT_NAME = 'Bait Al Zahabi'` — لا يتغير في أي لغة
- معرّف في `lib/i18n.js` ومستخدم في جميع الصفحات والمكونات

### 3. حذف لوحة admin
- مُحذوف نهائياً `/app/admin`
- **لوحة المدير** → `/manager`
- **لوحة رئيس النُدُل** → `/head-waiter`
- **واجهة النادل** → `/staff`

### 4. قسم أداء العمال — محسّن ومفهوم
- شرح مكتوب لكل مؤشر (طلبات، نداءات، تقييم)
- جدول يومي: يُظهر أداء كل نادل في كل يوم من الأسبوع
- شريط مرئي ملون لكل نادل
- إمكانية تصفية عرض نادل بعينه
- وضع "تصنيف الشهر" منفصل بترتيب واضح مع نسبة النمو

### 5. واجهة النادل — مُعاد بناؤها بالكامل
- تبويبات ثلاث: طلبات سريعة | رسالة مخصصة | سجل الطلبات
- 8 طلبات سريعة (بدلاً من 6) بألوان مختلفة
- زر النداء الرئيسي: حلقات نبض متحركة، 3 حالات (جاهز / جاري / تمت)
- تحذير واضح إذا لم يختر الزبون طاولة
- روابط سريعة لصفحات القائمة، الحساب، التقييم
- كامل الواجهة مترجمة (عربي/فرنسي/إنجليزي)

### 6. شعار جديد أرقى
- نجمة إسلامية ثمانية الأطراف داخل دائرة ذهبية
- حلقة خارجية تدور ببطء
- نقاط ذهبية في الأركان
- توهج ذهبي متدرج

### 7. Footer جديد فاخر
- شبكة معلومات 2×2 (موقع، ساعات، هاتف، إيميل)
- 5 أيقونات اجتماعية (Facebook، Instagram، Twitter، YouTube، TikTok)
- زخارف ذهبية وفواصل متحركة
- اسم المطعم بالإنجليزية والعربية معاً

### 8. الخلفية كاملة من أعلى لأسفل
- `background-attachment: fixed` يجعل الخلفية ثابتة وكاملة
- تدرجات ذهبية شفافة فوق خلفية داكنة
- لا مسافات بيضاء أو انقطاع في أي صفحة

## 🚀 تشغيل المشروع
```bash
npm install
# ضبط .env بقيمة DATABASE_URL الصحيحة
npx prisma generate
npx prisma db push
node prisma/seed.js
npm run dev
```
